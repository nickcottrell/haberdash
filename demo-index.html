üèóÔ∏è Architecture Overview

  • HTML: Semantic structure using native elements (<button>, <input type="radio">, etc.)
  • Base Layer: Structural CSS only - layout, spacing relationships, no colors
  • Theme Layer: Visual styling via CSS variables - colors, shadows, typography scales
  • Spectra System: Color-space encoding for dynamic theme generation (optional)
  • JavaScript: Behavior enhancement - auto-init, interactive components (optional)
